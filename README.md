# Insider Threat Detection System (Unsupervised ML)

## Overview
This project uses unsupervised machine learning to detect anomalous user behavior in healthcare systems.

## Motivation
Insider threats are difficult to detect because users already have authorized access. This project explores anomaly detection techniques to identify unusual patterns without labeled data.

## Features
- Behavior-based anomaly detection
- Identification of unusual login patterns
- Detection of abnormal data access
- Visualization of anomalies

## Model Used
Isolation Forest (Unsupervised Learning)

## Technologies
- Python
- Pandas
- Scikit-learn
- Google Colab

## Key Insight
The system detects anomalies without prior labeling, making it suitable for real-world cybersecurity applications.

## Future Improvements
- Real-time monitoring
- Integration with real-world system logs
- Advanced anomaly detection techniques
